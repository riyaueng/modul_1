Position

Was ist Position?

Mit CSS-Eigenschaften position wird gesteuer, wo die Elemente auf einer Seite dargestellt werden können.

Wie funktioniert das?

position legt fest wie Elemente in unserem Dokument positioniert werden. Die Eigenschaften sind top bottom left right

position: static

Standardwert für jedes HTML-Element. Das Element befindet sich an seiner Standardposition.

static

position: relative

Das Element bleibt an seiner ursprüngliche Stelle, kann aber von dort aus mit top oder left usw. verschoben werden.

relative

position: absolute

Das Element orientiert sich am nächstgelegenem nicht-statischen Eltern-Element, wenn es ein solches Element nicht gibt, dann orientiert es sich am HTML-Element.

Best practice: Das Eltern-Element sollte die Eigenschaft => position: relative erhalten.

absolute

position: fixed

Das Element orientiert sich am Browserfenster. Es wird aus dem Dokumentfluss entfernt und belibt beim Scrollen immer an seiner Position.

fixed

position: sticky

Das Element scrollt mit, bis eine seiner Kanten die durch top bottom left right angegebenen Korrdinate erreicht hat.

sticky
