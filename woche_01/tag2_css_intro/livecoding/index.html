Drei Möglichkeiten CSS in dein HTML einzubinden

1. Extern

In dem "head" kommt der Link mit Pfad zu der CSS Datei

Warum sollten wir diese Methode bevorzugen?

  • Schnellere Ladezeit: Es wird nur eine einzige CSS-Datei benötigt, um alle HTML-Dateinen zu gestalten. Mit weniger Codezeilen kann eine Website schneller laden. Die Website wird auch die CSS-Dateo für den nächsten Besuch Ihrer Besucher zwischenspeichern.
  • SEO-Verbesserung: Das Speichern von CSS-Stilen in einer anderen Datei macht die HTML-Dateo übersichtlicher und organsisierter. Als Ergebnis wird die Website einen sauberen und leichteren CODE haben, wodurch die Suchmaschinen sie besser crawlen können.
